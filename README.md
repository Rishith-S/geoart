## features
- dark mode ui
- uses coordinates or city names
- lots of themes like neon cyberpunk
- downloads hd pngs

## how to run
1. install dependencies
   ```bash
   npm install
   ```
2. start server
   ```bash
   npm start
   ```
3. open `http://localhost:3000` in your browser

## tech
- nodejs + express
- node-canvas for drawing
- vanilla js frontend
